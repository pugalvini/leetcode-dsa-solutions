## 🗂️ Structure

Problems are organized into topic-based folders (e.g., Arrays, Linked Lists, Trees). File names usually follow the format:  
`<problem_number>_<problem_title>.py`

Example:

```
leetcode-dsa-solutions/
│
├── README.md
├── arrays/
│   ├── 001_two_sum.py
└── graphs/
    └── 133_clone_graph.py
```
